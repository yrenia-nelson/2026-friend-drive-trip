# 2026 Friend Road Trip planning

This repository serves as the official planning workspace for the **2026 Coastal California & Pacific Northwest Road Trip** for five friends (Brian, Kenny, Jhon, Julius, and Melanie), traveling from October 10–21, 2026.

It is designed to be used with agentic coding assistants (such as Codex and Antigravity) to maintain a live, premium road trip itinerary and lodging plan. All cruise-specific materials and shore excursion data caches have been removed, transitioning this repository into a driving-only road trip itinerary.

## 🗺️ Key Workspaces & Planning Files

- **[Pacific Northwest Trip Itinerary.md](file:///Users/nelson/code/2026-friend-drive-trip/Pacific%20Northwest%20Trip%20Itinerary.md)**: The central road trip itinerary, covering day-by-day routing, driving times, food spots, active scenic adventures, and lodging tracks.
- **[AGENTS.md](file:///Users/nelson/code/2026-friend-drive-trip/AGENTS.md)**: Coordination rules and guidelines for coding agents editing this workspace.
- **[JOURNAL.md](file:///Users/nelson/code/2026-friend-drive-trip/JOURNAL.md)**: A running, dated change log recording all meaningful itinerary updates made by agents or humans.
