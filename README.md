# 2026 Cruise Drive Trip

This repo is a planning workspace for the 2026 Pacific Northwest road trip and Celebrity Edge Alaska cruise.

It is intended to be used with a coding agent such as Codex, Antigravity, or similar tools. The repo keeps trip notes, markdown itineraries, downloaded/sanitized travel data, and agent guidance together so future updates can be made by reading the local files instead of rediscovering context from chat history.

Start with:

- `Pacific Northwest Trip Itinerary.md` for the main trip plan.
- `data/celebrity-shorex/` for Celebrity Cruises shore excursion data and analysis.
- `AGENTS.md` for notes useful to coding agents, including how the Celebrity API data was captured and what must not be committed.
